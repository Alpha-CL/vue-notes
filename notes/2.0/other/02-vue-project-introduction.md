#### scoped

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 使用 scoped 时，父组件的样式不会泄漏到子组件中
 * 但是，子组件的根节点将同时受父组件的作用域CSS和子组件的作用域CSS的影响
 * 
 * 
 ** data-v-[hash]: 为了便于在父组件中 可调整 子组件根元素的 样式及布局
 *                 使用深度选择器，编译后会利用属性选择器，将 data-v-[hash] 作为选择选择器的属性 
 *                 从而可在 父组件中 调整子组件根元素的样式及布局
 */


<FatherComp data-v-[hash]>

    <SonComp data-v-[hash] class="son"/>
    
</FatherComp>


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### deep selector

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 若希望 scoped 样式选择器是“较深的”，即影响子组件，则可以使用>>>组合器

    .a >>> .b { /* ... */ }
    
</style>

// complie

}


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
