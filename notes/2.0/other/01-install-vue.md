## install vue

#### install vue cli

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * Install Vue 
 * 
 * 
 * 8.11.0+
 */

npm install -g @vue/cli                         // vue 命令行工具

npm install -g @vue/cli-service-global          // 快速原型开发工具


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### vue module scoped

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * scoped
 * 
 * 
 * 带有 scoped 特性的标签仅作用域当前的 component
 */

<template scoped>

    // hteml

</template>


<script scoped>

    // js

</script>


<style scoped>

    // css

</style>


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### create vue app

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * command create vue
 * 
 * 
 * 
 */


vue create [name]


* space === check
* return === confirm


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
