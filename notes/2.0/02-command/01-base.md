## v-bind

#### v-bind === :

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * v-bind: === :
 *
 *
 * 动态地将一个或多个属性或组件属性绑定到表达式
 *
 ** 绑定的属性对应的属性值，可以使用 js (this -> vm)
 ** 绑定 class & style 时，可支持其他类型( eg: array, object ... )
 */


// 绑定单个 html属性
<div v-bind:[attribute]="vuejs statement"/>          // [attribute]: html属性名


// 绑定 动态属性名
<div v-bind:[key]="vuejs statement"/>                // [key]: 属性变量名


// 语法糖: v-bind: === :
<div :[attribute]="vuejs statement" />


// 同时绑定多个 html属性对象: 此时 class 和 style 绑定不支持数组和对象
<div v-bind="{class: 'custom-class', 'custom-prop': 'value', 'font-size': '12px'}"/>


// 将当前组件的 props 传递给子组件
<childComp v-bind="$props" />


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

## v-if statement & v-show

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * v-if
 *
 * 
 * v-if 无法添加到 <template/> 标签上，因为编译后该标签不存在
 * v-if & v-else-if & v-else 书写的元素必须连续，否则不被识别
 */


<div v-if="condition">a</div>
<div v-else-if="condition">b</div>
<div v-else>not a/b</div>



///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```

#### v-if vs v-show

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * v-if statement vs v-show
 *
 *
 ** 实际开发中，应根据实际情况选择使用
 */


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


//v-if

    - 语句能够控制是否生成 vnode( 间接控制了是否生成对应的 dom )

    - 只有条件为真时，才创建 虚拟节点，从而创建对应的 真实节点元素

    - <div v-if="visibility: boolean">show or hide</div>

        - true: <div>show or hide</div>

        - false: no dom

//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


// v-show

    - 始终生成 vnode( 间接导致了始终生成对应的dom )

    - 主要控制指定元素的 display 属性( 无法减少节点的渲染量 )

    - <div v-show="boolean">show or hide</div>

        - true: <div style="display: block">show or hide</div>

        - false: <div style="display: none">show or hide</div>
        

///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
