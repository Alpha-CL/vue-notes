#

#### BEM

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * BEM
 * 
 * 
 * 自定义规范命名类样式
 */

block       // 块

    - 独立的实体，本身具有意义
      ( 类似一块积木，一个块既可以是"简单的"也可以是 "复合的(包含其他块)" )

element     // 元素

    - 块的一部分，没有独立含义，并在语音上与该块相关联

modifier    // 修饰符

    - 块或元素上的标志，用来描述改变该块或元素的行为


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


    // state_01                            // state_02
    
    + ---- warp --------- +                + ---- wrap -------- +
    |                     |   click ele    |                    |
    |   btn--default ...  | -------------> |   btn--active ...  |
    |                     |                |                    |
    + ------------------- +                + ------------------ +
    
    * 点击按钮后，按钮状态发生改变

eg: class:      .block-element--modifier

    block:      wrap
    element:    btn
    modifier:   default, active


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
