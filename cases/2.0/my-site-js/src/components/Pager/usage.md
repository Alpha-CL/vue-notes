#### Pager usage

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


<Pager

    // props

    current?: number = 1;                   // 当前页码
    total?: number = 0;                     // 总数据量
    limit?: number = 10;                    // 页容量
    visibleNumber?: number = 10;            // 可见页码数量
    
    
    // methods
    
    pageChange(nextPageNumber: number);     // 切换页码
    
/>


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
